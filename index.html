Mailing list upload

Please note: The exact format of the CSV files is very important. Some of the columns can be blank, but the column must be there.

The Property Owners file should have the following 11 columns, in this order:

  1. Folio
  2. Civic
  3. Name 1
  4. Name 2
  5. Mailing
  6. Total Assess
  7. Included Assess
  8. Ann Chg
  9. Unit
  10. House
  11. Street

The Business Licenses file should have the following 15 columns, in this order:

  1. RECORD
  2. LICENSE NUMBER
  3. ADDRESS
  4. LICENSE TYPE
  5. STATUS
  6. LICENSE YEAR
  7. BUSINESS NAME
  8. BUSINESS TRADE NAME
  9. DATA FROM
  10. MAIL ADDRESS1
  11. MAIL ADDRESS2
  12. MAIL ADDRESS3
  13. MAIL ADDRESS4
  14. WORK PHONE1
  15. WORK PHONE2
